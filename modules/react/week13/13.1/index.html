# Setting Up for Tailwind in React

  1. First we need to create our React project with vite, as per normal.
npm create vite@latest project-name
npm install
1
2
  1. Install Tailwind CSS and its Peer Dependencies:

Second, we'll need to install Tailwind CSS, PostCSS, and Autoprefixer as development dependencies. Open your terminal in your project directory and run:

npm install -D tailwindcss postcss autoprefixer
1
  1. Initialize Tailwind CSS:

Next, initialize Tailwind CSS to generate tailwind.config.js and postcss.config.js files.

npx tailwindcss init -p
1

This command will create the following files in your project's root:

  • tailwind.config.js: Configuration file for customizing Tailwind CSS.
  • postcss.config.js: Configuration file for PostCSS, which Tailwind CSS uses.
  1. Configure Template Paths:

We need to tell Tailwind CSS where to look for your template files (your React components). Update tailwind.config.js to include the paths to our JSX/TSX files.

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}'],
  theme: {
    extend: {},
  },
  plugins: [],
};
1
2
3
4
5
6

  • ./index.html: Includes your main HTML file.
  • ./src/**/*.{js,ts,jsx,tsx}: Includes all JavaScript, TypeScript, and JSX/TSX files within our src directory. Adjust these paths if our project structure is different.
  1. Add Tailwind Directives to Our CSS:

Create a global CSS file (or use an existing one) and add the Tailwind CSS directives. Typically this is done in the index.css file. If our project does not have one create it in the src folder.

/* ./src/index.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
1

4

Import the global CSS file into your main entry point (src/main.jsx or src/main.tsx). This should already be done, but it is important to keep this.

// src/main.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import './index.css'; // Import your Tailwind CSS file

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

Now, when we start our Vite development server with npm run dev, the Tailwind classes will be available for use.

For example, in the App.js file.

// src/App.jsx
export default function App() {
  return (
    <div className="flex items-center justify-center h-screen bg-gray-100">
      <h1 className="text-3xl font-bold text-blue-600">Hello, Tailwind CSS!</h1>
    </div>
  );
}
