# Self-Directed

Beyond the summer course, you can also do your own investigations into topics based on your own interests.

Here are a few suggestions to get you started.

  • HTML5 Canvas. There is an HTML tag <canvas> which can be used as a 2d or 3d drawing surface. It can be used to display video frames. It can be used to manipulate image data directly too. MDN Canvas guide (opens new window) YouTube short intro tutorial on canvas (opens new window).

  • SVG animations. SVG is the XML-based image format. You can couple CSS with these elements to create animation. There are also libraries for animating SVG images. Snapsvg.io (opens new window) is a good first one.

  • Web Authentication is the latest approach to authentication for the web. Heard of tokens? Of course, you have used theme in your headers. How about passkeys? You may have come across those on a website, but how do they work? https://developer.mozilla.org/en-US/docs/Web/API/Web_Authentication_API (opens new window) is the MDN guide to understanding the Web Authentication API and passkeys. Here are a couple Google guides to passkeys - https://developers.google.com/identity/passkeys (opens new window) and https://safety.google/authentication/passkey/ (opens new window)

  • Audio is a big area. You can start delving into the Web Audio API on MDN (opens new window).

  • Streaming and WebRTC are related topics. Streaming is a way to handle any kind of data. You can stream content from a server to a client. WebRTC is the Web Real Time Communications API. It is the way to create Peer to Peer streaming of audio and video. Build your own Zoom.

  • If you want to learn more about databases, look into Relational databases such as MySQL, PostgresSQL, and SQL Server. Then look at Vector databases and look at how vector databases are being built within the Relational and Document databases.

  • Proxy objects in Javascript MDN Proxies (opens new window).

  • More Web APIs? There is a long list of Web APIs that we can control with JavaScript. Some are fully supported and some are still experimental. Have a look at the list - https://developer.mozilla.org/en-US/docs/Web/API (opens new window)

  • Want to be a better programmer in general? Try to get a better understanding of data structures and algorithms (opens new window).

  • Get stuck while learning these things on your own? This is a perfect time to use the AI Chat tools - ChatGPT, Gemini, Claude, Perplexity, etc. They are like having a personal tutor who can explain any topic in simpler terms.

  • Pick a couple topics that are of personal interest to you and try building something with those technologies. What you build doesn't matter. Building is learning. It is the practice that will make you a better developer and designer.
